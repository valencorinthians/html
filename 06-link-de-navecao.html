Link para site Externo

link externo aponta para um endereço (URL) fora de seu site ou servidor onde seus arquivos estao hospedados

como ele funciona?

para que o navegador saiba que seu dominio, o atributo href/do elemento <a>deve conter o endereço completo do site sxterno, incluindo o protocolo (http:// ou https://).

Atributo target="_blank"

O atributo target="blank" é usado para abrir o link em uma nova aba ou janela, permitindo que o usuário mantenha a página atual aberta enquanto visita o site externo.
